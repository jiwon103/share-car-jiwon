# Rental Car Service Test Website

This project is a prototype website designed to demonstrate a real-time car rental service.

## Key Features
- **Auto-fading Main Banner:** Smooth transition between promotional car images.
- **Real-time Vehicle List:** Fetches and displays available cars from a linked Google Spreadsheet.
- **Vehicle Detail Modal:** Provides specific car information when selected.
- **Consultation Link:** Direct connection to Kakao OpenTalk for customer inquiries.
